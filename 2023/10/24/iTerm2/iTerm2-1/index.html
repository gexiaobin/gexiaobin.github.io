iTerm2安装配置

1 软件安装

1.1 Homebrew安装

打开电脑终端，命令行输入一下命令，完成iTerm2安装

如提示未安装Homebrew，请先安装Homebrew

2 基本设置

2.1 设置iTerm2为默认终端

打开iTerm2，左上角选择iTerm2，点击“Make item2 Default Term”即可

3 安装oh-my-zsh

3.1 关于oh-my-zsh

mac下默认的Shell是bash，虽然bash的功能已经很强大了，但是zsh拥有更多的自定义空间，并支持各种丰富的扩展，可以实现更强大的命令补全，命令高亮，自动跳转等一系列功能，这些功能极大的提升了我们操作命令行的效率。

当然使用zsh Shell 的代价是牺拖慢了一点点启动速度，不过这点几乎可以直接忽略不计，因为默认的 zsh 配置起来比较麻烦，一个叫 robbyrussel 的用户在 GitHub 上制作了一个配置文件 oh-my-zsh，该文件逐步的被大家接受，目前也是公认的较为流行的zsh 配置方式。
